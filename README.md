`ViewCompose` 是一个基于 Android View 渲染管线的声明式 UI 框架，采用 Kotlin DSL 描述 UI，并通过可观测状态驱动增量更新。  
`ViewCompose` is a declarative UI framework on top of the Android View rendering pipeline. It uses a Kotlin DSL to describe UI and applies incremental updates from observable state.

## 架构总览 | Architecture Overview

### 渲染链路 | Rendering Flow

1. DSL 构建 `VNode(NodeSpec)` 树。  
   DSL builds a `VNode (NodeSpec)` tree.
2. Runtime 进行状态追踪与增量重组（SlotTable Lite）。  
   Runtime tracks state and performs incremental recomposition (SlotTable Lite).
3. Renderer 执行 diff / patch，将语义映射到 Android View。  
   Renderer performs diff/patch and maps semantics to Android Views.
4. Host 管理会话生命周期，并注入平台环境（Lifecycle/ViewModel/FrameClock）。  
   Host manages render-session lifecycle and injects platform environment (Lifecycle/ViewModel/FrameClock).

| `viewcompose-widget-core` | DSL、Theme、Defaults、Locals。 / DSL, Theme, Defaults, Locals. |

## 快速开始 | Quick Start

### 环境要求 | Requirements

- JDK 17
- Android SDK 36（`compileSdk = 36`）
- Android Studio（推荐最新稳定版）
- 运行 `qaFull` 需要连接设备或模拟器  
  Running `qaFull` requires a connected device or emulator.

### 拉取与构建 | Clone & Build

```bash
git clone https://github.com/guozhiqiang123/ViewCompose.git
cd UIFramework
./gradlew qaQuick
```

### 质量门禁命令 | Quality Gates

```bash
# 编译 + 单测 + 模块边界检查
./gradlew qaQuick

# 预览快照回归（Paparazzi）
./gradlew qaPreview

# 全量（qaQuick + instrumentation）
./gradlew qaFull
```

### 运行 Demo | Run Demo

1. 用 Android Studio 打开项目。  
   Open the project in Android Studio.
2. 运行 `app` 模块的 `MainActivity`。  
   Run `MainActivity` in the `app` module.
3. 在 Demo 首页按模块浏览能力（Foundations / State / Layouts / Animation / Graphics 等）。  
   Browse capabilities by module on the demo home (Foundations / State / Layouts / Animation / Graphics, etc.).

## 最小接入示例 | Minimal Integration Example

```kotlin
class SampleActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setUiContent { _ ->
            UiTheme {
                Column(spacing = 12.dp, modifier = Modifier.padding(16.dp)) {
                    Text(text = "Hello ViewCompose")
                    Button(text = "Click Me", onClick = { /* handle click */ })
                }
            }
        }
    }
}
```

核心入口 API 在 `viewcompose-host-android`：`ComponentActivity.setUiContent(...)` 与 `Fragment.setUiContent(...)`。  
Core entry APIs are in `viewcompose-host-android`: `ComponentActivity.setUiContent(...)` and `Fragment.setUiContent(...)`.
